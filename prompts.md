# Reusable AI Prompts — UGC Ad Content System
**Use these prompts in Claude, ChatGPT, or Gemini for any brand**

---

## Prompt 1 — Hook Generator

```
You are a UGC ad scriptwriter for local businesses and D2C brands.

Brand: [BRAND NAME]
Product/Service: [SERVICE NAME]
Core benefit: [MAIN BENEFIT]
Target audience: [AUDIENCE DESCRIPTION]
Location: [CITY NAME]
Platform: [Instagram Reels / YouTube Shorts / Stories]

Generate 5 high-converting hooks for a UGC-style video ad.

Rules:
- Sound like a real customer talking, NOT a brand
- No salesy language or buzzwords
- Each hook must be under 15 words
- Use one of these hook types per hook:
  1. Problem-led
  2. Social proof
  3. Transformation
  4. Contrarian
  5. Relatable moment (POV)

Output format:
Hook 1 (type): [text]
Why it works: [1 sentence]
```

---

## Prompt 2 — Full Script Generator

```
You are a UGC ad scriptwriter. Write a 30-second video ad script.

Brand: [BRAND NAME]
Service: [SERVICE NAME]
Hook to use: [PASTE HOOK HERE]
Audience pain point: [SPECIFIC PROBLEM]
Key proof point: [RESULT / INGREDIENT / REVIEW COUNT]
CTA: [DESIRED ACTION]
Location: [CITY]

Framework: Hook → Problem → Agitate → Discovery → Proof → Soft CTA

Rules:
- First person, conversational tone
- No scripted-sounding lines
- Include visual direction in [brackets]
- Keep each section under 2-3 lines
- End with a soft CTA, not a hard sell
- Sound like a real customer — not a brand ambassador

Output: Timestamped script with section labels
```

## Prompt 3 — Caption + Hashtag Generator

```
Write 2 Instagram captions for a UGC ad video.

Brand: [BRAND NAME]
Service: [SERVICE NAME]
Core message: [1-line summary of the video]
Tone: [honest / educational / punchy / emotional]
Location: [CITY]
Promo (if any): [OFFER OR CODE]

Caption 1: Storytelling style (4-6 lines + hashtags)
Caption 2: Short punch style (2-3 lines + hashtags)

Rules:
- Sound like a real person, not a brand page
- Include 8-10 niche + local hashtags at the end
- Mention price if under ₹2,000
- Add location (area + city) in the caption body
```

## Prompt 4 — Platform Adapter

```
Take this UGC script and adapt it for 3 platforms:

Original script: [PASTE SCRIPT HERE]

Adapt for:
1. Instagram Reels (30 sec, casual, fast cuts)
2. Instagram Stories (15 sec, ultra punchy, swipe-up CTA)
3. YouTube Shorts (45 sec, slightly more educational)

For each version:
- Adjust length and pacing
- Change CTA to match the platform
- Keep the same hook and core message
- Note any visual direction changes
```
