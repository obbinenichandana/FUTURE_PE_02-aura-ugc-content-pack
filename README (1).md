## What Is This?

This is a complete UGC (User Generated Content) ad content pack created for **Aura Skin & Hair Studio**, a real salon in Governorpet, Vijayawada, Andhra Pradesh.

It was built using AI tools (Claude by Anthropic) as part of a content marketing task to demonstrate how UGC-style ad scripts can be created for local businesses.

## How to Use These Prompts

1. Open `prompts.md`
2. Copy any prompt
3. Paste into Claude (claude.ai) or ChatGPT
4. Fill in the `[BRACKETED]` fields with your brand details
5. Generate hooks and scripts instantly

These prompts work for **any local business** — just swap the brand name, service, and city.
